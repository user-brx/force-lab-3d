⛸️ Patinadores

Conservação do momento linear e a relação entre força, massa e aceleração.

O conceito

Dois patinadores se empurram. A força é idêntica nos dois (3ª lei), mas como a = F/m, o mais leve sai mais rápido. Num sistema isolado (sem atrito), o momento total se conserva e permanece zero: os dois momentos são iguais e opostos. Com atrito (força externa), o momento total deixa de se conservar e ambos param.

As fórmulas

FA→B = − FB→A   ⇒   mA·vA + mB·vB = 0 Mesma força e mesmo tempo ⇒ impulsos opostos ⇒ momentos opostos que somam zero.
vA / vB = mB / mA As velocidades ficam na razão inversa das massas: o leve dispara, o pesado mal anda.
Impulso: J = F·Δt = Δp A mesma força aplicada pelo mesmo tempo dá o mesmo módulo de momento a cada um.

Exemplo resolvido

Empurrão de 300 N por 0,4 s entre azul (60 kg) e vermelho (90 kg):

Impulso = F·Δt = 300·0,4 = 120 N·s em cada um.

vazul = 120/60 = 2,0 m/s; vvermelho = 120/90 = 1,33 m/s.

Momento total = 60·2,0 − 90·1,33 = 120 − 120 = 0 ✓ (conservado, no vácuo/gelo sem atrito).

No simulador, no gelo/vácuo o momento total permanece zero "para sempre"; no asfalto o atrito (força externa) freia os dois e o total deixa de ser zero - ótima oportunidade para discutir o que é um sistema isolado.
