✈️ Avião

Sustentação, arrasto induzido e parasita, e a barreira do som (arrasto de onda).

O conceito

A hélice/turbina joga ar para trás e o ar empurra o avião para frente (3ª lei). As asas defletem o ar para baixo (downwash) e o ar empurra as asas para cima: é a sustentação. Tudo depende do ar - sem atmosfera o avião não voa (contraste direto com o foguete). Perto de Mach 1 surge o arrasto de onda, a famosa "barreira do som".

As fórmulas

L = ½ · ρ · v² · S · CL    D = ½ · ρ · v² · S · CD L = sustentação, D = arrasto; S = área da asa; CL, CD = coeficientes.
CD = CD0 + k·CL² + CD,onda(Mach) Parasita (CD0) + induzido (∝ CL²) + arrasto de onda (dispara perto de Mach 1).
vestol = √( 2·m·g / (ρ·S·CL,máx) ) Velocidade mínima para a sustentação igualar o peso. Como v ∝ 1/√ρ, no ar rarefeito (Marte) precisa-se de muito mais velocidade.
Mach = v / a(h)     a(h) = a₀·√( T(h)/T₀ ) A velocidade do som cai com a temperatura (lapse rate ISA até a tropopausa).

Exemplo resolvido

Velocidade de estol (3000 kg na Terra): vestol = √(2·3000·9,81 / (1,225·30·1,5)) = √(58 860 / 55,1) ≈ 32,7 m/s (~118 km/h).

Em Marte (ρ = 0,02 kg/m³) o mesmo avião precisaria de v ∝ 1/√ρ ≈ 7,8× mais velocidade - praticamente impossível com asas normais. No ar denso de Vênus (ρ = 65) é o oposto: voa quase parado.
